COMPARISON PARAGRAPH:

By examining the alogrithms, it seems that, if possible, sometimes rand
behaves slightly better than other algos, but that is only if it gets 
lucky. FIFO seems to perform well as mem increases, LRU seems to be 
neutral, but does perform better as mem increases, same with CLOCK. 
But opt seems to perform well, since its the algorithm that chooses 
the 'best' page to evict.

LRU ANALYSIS:

As mem size increases LRU seems to perform better. It gets 
increasingly well, better than FIFO and CLOCK, due to it taking 
advantage of memory, more mem more pages, and LRU evicts pages based
on timestamp, this it gets more and more efficient. This lru has a 
higher hitrate as mem size increases.
